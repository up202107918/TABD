Place application screenshots here for the documentation and report.

## File Naming Convention

Use descriptive names in lowercase with underscores:
- ✅ Good: `homepage_with_map.png`
- ❌ Bad: `Screenshot 2026-06-01 at 3.45.23 PM.png`

## Image Format

- **Preferred**: PNG (lossless, good for UI)
- **Acceptable**: JPG (for photos/presentations)
- **Resolution**: At least 1920x1080 for desktop views
- **Size**: Keep under 2MB per screenshot
